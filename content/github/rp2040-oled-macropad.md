## Objective & Design Thoughts
The goal of this project is to build a highly responsive, custom hardware macropad using the Raspberry Pi RP2040 chip. The integrated OLED display provides real-time layer feedback, active profile names, and keypress visualizations.

Using CircuitPython or MicroPython keeps the software highly customizable without compiling C code every time a keymap changes.

## Production Status
- **Hardware:** RP2040 Board (Pi Pico / KB2040) + I2C SSD1306 OLED Display + Mechanical Keyswitches.
- **Software:** Python scripts running locally on the micro-controller.
